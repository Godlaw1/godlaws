# Contributing to Godlaws

Thanks for your interest in contributing to Godlaws Foundation!

## Our Projects

- **Apex Fractal Bot** — Automated futures trading bot
- **Copy Trader** — Multi-broker trade replication system

## How to Contribute

1. Fork the project repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test thoroughly (backtest for strategy changes, dry-run for broker changes)
5. Commit: `git commit -m "Add your feature"`
6. Push: `git push origin feature/your-feature`
7. Open a Pull Request

## Guidelines

- Keep PRs focused on a single change
- Never commit credentials, API keys, or secrets
- Follow the existing code style
- Include test results for trading strategy changes
- Document new broker integrations

## Reporting Issues

Open an issue with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Relevant logs (redact any credentials)

## Security

If you discover a security vulnerability, please report it privately to the maintainers. Do not open a public issue.

## Code of Conduct

Be respectful and constructive. We're building open financial tools for everyone.
