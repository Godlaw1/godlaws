# Godlaws Foundation

Open source trading tools and financial technology for everyone.

## Mission

Godlaws is a non-profit foundation dedicated to making professional-grade trading tools accessible to all. We believe financial technology should be open, transparent, and community-driven.

## Projects

| Project | Description | Status |
|---------|-------------|--------|
| [Apex Fractal Bot](https://github.com/godlaws/apex-fractal-bot) | Automated futures trading bot using fractal pullback strategy | Active |
| [Copy Trader](https://github.com/godlaws/copy-trader) | Multi-broker trade replication system (9 brokers supported) | Active |

## What We Offer

- **Open Source Trading Bots** — Production-ready automated trading systems
- **Strategy Frameworks** — Backtesting and optimization engines
- **Multi-Broker Integration** — Connect to cTrader, MetaTrader, OANDA, Alpaca, Binance, IB, NinjaTrader
- **Education** — Learn algorithmic trading through real, working code
